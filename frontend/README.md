# Virtual Hair Try-On Frontend

This is the frontend for the Virtual Hair Try-On application built with React.js, Bootstrap, and Axios.

## Features

- Select hair length (Short, Medium, Long)
- Browse hairstyles based on selected length
- Upload user photo or capture from camera (placeholder)
- Preview hairstyle overlay on user photo
- Adjust position, scale, and rotation of hairstyle
- Save previews

## Setup Instructions

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173` (default Vite port).

## Backend

Make sure the backend is running on `http://localhost:1200`. Refer to the backend README for setup.

## Technologies Used

- React 19
- Bootstrap 5
- Axios for API calls
- React Router DOM (for potential future routing)
